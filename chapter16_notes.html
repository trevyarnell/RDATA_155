2 What is ggplot2?

ggplot allows me to create plots using a data frame. Common patterns are ggplot(data = df, mapping = aes(x, y, other aesthetics)) or ggplot(data = df) or ggplot. The first one is useful if all layers of the plot use the same data and aesthetics. The second pattern does not define aesthetics upfront, useful if using multiple datasets. The third one is used when creating complicated graphics. data = and mapping = are not needed however can be added for clarity.

4 Mapping Aesthetics

Aesthetics are the visual properties of your graph and are done throught aes arguement.

Mapping divorce rate to margarine consumption

divorce_margarine %>%
  ggplot(aes(x = divorce_rate_maine, y = margarine_consumption_per_capita))

The plot is blank because we haven’t added any geometric objects to the plot yet. The aes() step needs to come first before we add geoms’.

5 Scatterplots

geoms are the shapes that are drawn on the plot, points, lines, bars, boxes, boxplots, etc.

?geom_point

Typical Syntax: geom_point(mapping = NULL, data = NULL, ..., na.rm = FALSE, show.legend = NA, inherit.aes = TRUE)

Add points using geom_point

divorce_margarine %>%
  ggplot(aes(divorce_rate_maine, margarine_consumption_per_capita)) +
  geom_point()
divorce_margarine %>%
  ggplot(aes(divorce_rate_maine, margarine_consumption_per_capita)) +
  geom_point(aes(color = year))

Creates a blue gradiant corresponding to the year of the data point. By using aes() in geom_point I am defining the specfic aesthetics assigned to that layer of the plot.

divorce_margarine %>%
  ggplot(aes(divorce_rate_maine, margarine_consumption_per_capita)) +
  geom_point(aes(color = year, size = margarine_consumption_per_capita))
divorce_margarine %>%
  ggplot(aes(divorce_rate_maine, margarine_consumption_per_capita)) +
  geom_point(aes(color = year)) +
  geom_smooth(method = "lm", se = FALSE)
`geom_smooth()` using formula = 'y ~ x'

6 Histograms

set.seed(2025)
Tallness <- data.frame(
  Country = c(rep("USA", 1000), rep("CAN", 1000)),
  Height = c(rnorm(1000, 67, 2.5), rnorm(1000, 66, 1.5))
)
nrow(Tallness)

Tallness %>%

             mean_height = mean(Height),
             sd_height = sd(Height),
             .groups = "drop")

Build a histogram

Tallness %>%
  ggplot(aes(Height)) +
  geom_histogram()
`stat_bin()` using `bins = 30`. Pick better value `binwidth`.
Tallness %>%
  ggplot(aes(Height)) +
  geom_histogram(fill = "steelblue",
                 color = "white",
                 binwidth = .25)

6.1 Density Plot

Create a density plot and fill the area under the curve and add transparency.

Tallness %>%
  ggplot(aes(Height)) +
  geom_density(fill = "purple",
               alpha = 0.4,
               color = "purple",
               linewidth = 1)

6.2 Overlaid Histograms

tall_compared_fig <- Tallness %>%
  ggplot(aes(Height, fill = Country)) +
  geom_histogram(position = "identity",
                 alpha = 0.6)

print(tall_compared_fig)
