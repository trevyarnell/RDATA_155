2 Counts, Center, and Spread

Numeric variable: numbers that are measured or calculated Categorical variable: stores labels or group names

Count: for a whole dataset means the total number of rows; grouped data mean the number of rows in each group

Center: typical value Mean: average Median: the middle value

Spread: how much the values vary Standard deviation: how far values vary from the mean Range: distance from the smallest to largest value

3 count()

3.1 Counting one varible

mtcars %>%
  count(cyl)
  cyl  n
1   4 11
2   6  7
3   8 14

Returns how many cars in the mtcars dataset have 4, 6, and 8 cylinders grouped by cylinder count.

3.2 count() versus length()

length() counts how many values are in a vector count() counts the rows in a df for each category or combination of categories

3.3 count() with sorting

mtcars %>%
  count(cyl, sort = TRUE)
  cyl  n
1   8 14
2   4 11
3   6  7

Sorts the groups with the most vehicles to the least

4 summarize()

mtcars %>%
  summarize(mean_mpg = mean(mpg))
  mean_mpg
1 20.09062

Creates a new column that has the mean MPG for all the cars in the mtcars dataset

mtcars %>%
  summarize(mean_mpg = mean(mpg),
            max_mpg = max(mpg),
            count = n())
  mean_mpg max_mpg count
1 20.09062    33.9    32
