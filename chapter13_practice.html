6 National mean counts by disease

In this task I am calculating the total mean of the count for each disease and storing that in Infectious_Burden

Infectious_Burden <- us_contagious_diseases %>%
  group_by(disease) %>%
  summarize(
    mean_count = mean(count, na.rm = TRUE),
    .groups = "drop"
  ) %>% 
      arrange(desc(mean_count))

7 Reflection

When using group_by() before summarize() it allows you to get descriptive statistics from different groups within the data. If you were to only use summarize() on a data set it would give you the mean, sd, etc for all rows and it would not be broken down by groups. For example earlier in this chapter I wrote code that calculated the. average MPG for all vehicle in the mtcars dataset shortly after that I create code that showed the average MPG by number of cylinders.
