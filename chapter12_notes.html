3 Vectorized Conditionals

3.1 ifelse()

Code 
x <- c(3, 2, 3, 6, 5)
Code 
length(x)
[1] 5

x is a vector with the length of 5 rows.

Code 
ifelse(test = x > 3, yes = "Large", no = "Small")
[1] "Small" "Small" "Small" "Large" "Large"
Code 
length(ifelse(x > 3, "Large", "Small"))
[1] 5

3.2 case_when()

Code 
case_when(c(3, 2, 3, 6, 5) > 3 ~ "Large", .default = "Small")
[1] "Small" "Small" "Small" "Large" "Large"

Don’t forget to load any required libraries when trying to use functions no in base R.

3.3 More than Two Categories

Code 
scores <- c(55, 65, 75, 85, 95)

3.3.1 Nested ifelse()

Code 
ifelse(scores < 60, "Low",
       ifelse(scores < 80, "Medium", "High"))
[1] "Low"    "Medium" "Medium" "High"   "High"

3.3.2 Multiple Rule case_when()

Code 
case_when(scores < 60 ~ "Low",
          scores < 80 ~ "Medium",
          TRUE ~ "High")
[1] "Low"    "Medium" "Medium" "High"   "High"
Code 
case_when(scores < 80 ~ "Medium",
          scores < 60 ~ "Low",
          TRUE ~ "High")
[1] "Medium" "Medium" "Medium" "High"   "High"

4.1.1 mutate() with Arithmetic

Code 
measurements <- data.frame(height_in = c(60, 65, 70))
measurements
  height_in
1        60
2        65
3        70

The measurements data frame has one column called height_in.

Code 
measurements %>%
  mutate(height_cm = height_in * 2.54)
  height_in height_cm
1        60     152.4
2        65     165.1
3        70     177.8

4.1.2 case_when() inside mutate()

Code 
student_scores <- data.frame(scores = c(55, 68, 72, 83, 91))
Code 
student_scores <- student_scores %>%
  mutate(
    score_level = case_when(
      scores < 60 ~ "Low",
      scores < 80 ~ "Medium",
      TRUE ~ "High"
    )
  )
Code 
head(student_scores)
  scores score_level
1     55         Low
2     68      Medium
3     72      Medium
4     83        High
5     91        High

student_scores is a fataframe with columns names: scores, score_level.

4.2 Overwrite Column Values

Code 
unique(mtcars$am)
[1] 1 0
Code 
cars_overwrite <- mtcars %>%
  mutate(
    am = case_when(
      am == 0 ~ "Automatic",
      TRUE ~ "Manual"
    )
  )
Code 
unique(cars_overwrite$am)
[1] "Manual"    "Automatic"

The am column in cars_overwrite has the unique values: Manual, Automatic.
