I am attempting to label the height in the heights dataset as “Tall” if it is greater than 68 and otherwise “Not Tall”

Code 
height_label <- ifelse(
  test = heights$height > 68,
  yes = "Tall",
  no = "Not Tall"
)

     508      542

4 Recode with mutate() and case_when()

I am labeling the mpg of cars from the mtcars dataset as either “High MPG” if it is greater than 25, “Low MPG” if it is less than 15, and otherwise “Medium MPG”.

Code 
mpg_recode <- mtcars %>%
  mutate(mpg_class = case_when(
    mpg > 25 ~ "High MPG",
    mpg < 15 ~ "Low MPG",
    TRUE ~ "Medium MPG"
  ))
Code 
table(mpg_recode$mpg_class)

  High MPG    Low MPG Medium MPG 
         6          5         21

There are 6 High MPG cars, 5 Low MPG cars, and 21 Medium MPG cars.

I am recording the temperature variable in the stars dataset to indicate if the star is “Cool”, Warm, or “Hot”.

Code 
stars_recode <- stars %>%
  mutate(temp_group = case_when(
    temp < 4000 ~ "Cool",
    temp <= 7500 ~ "Warm",
    TRUE ~ "Hot"
  ))
